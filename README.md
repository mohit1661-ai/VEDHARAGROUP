- Strategic CTA placement

## 🚀 Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/mohit1661-ai/VEDHARAGROUP.git

# Navigate to project
cd VEDHARAGROUP

# Open in browser
open index.html
```

### Development

```bash
# Install dependencies (if using Node/npm)
npm install

# Start development server
npm start

# Build for production
npm run build
```
